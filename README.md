# SensorApp

SensorApp

Access and acquiring raw sensor data of following sensors:

Motion sensors:
    - accelerometer
    - accelerometer uncalibrated
    - gravity
    - gyroscope
    - gyroscope uncalibrated
    - linear acceleration
    - rotation vector
    - step counter

Position sensors:
	  - game rotation vector
    - geomagnetic rotation vector
    - magnetic field
    - magnetic field uncalibrated
    - proximity

Environment sensors:
    - ambient temperature
    - light
    - pressure
    - relative humidity
